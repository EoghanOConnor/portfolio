Facial Verification

A Siamese CNN network is trained on using LFW dataset. A Siamese network is a type of neural network with two identical subnetworks that are used to compare the similarity between two inputs. The subnetworks share weights and architecture, allowing the network to learn a feature representation that is invariant to small variations in the input data, making it useful for tasks such as image matching and face recognition.

This is a Facial Identification and Verification tasks

ResNet and DenseNet Object Classifier

ResNet and DenseNet model for object classification for the CIFAR-10 Dataset. ResNet is a neural network architecture that uses skip connections to allow gradients to flow more easily during training, enabling deeper networks to be trained. DenseNet is a neural network architecture that connects all layers directly to each other, resulting in dense connections that allow information to flow more easily between layers.
