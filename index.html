Ultrasonic controlled servo motor

Ultrasonic sensors are used to measure the distance to an object by emitting ultrasonic sound waves. These sound waves are reflected off the object and are captured by the sensor. The recieved wave is converted into an electrical signal and provides measurement of that distance. This signal can be used to control other components attached to a mircocontroller. These sensors are used automobilie self-parking technology and obstacle detection systems

This is a software embedded project. Using HC-SR04 Ultrasonic sensor and a servomotor
